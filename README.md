# Three-tier-aws-terraform
🔹 Project Overview
This repository contains a complete three-tier architecture on AWS built with Terraform.
It automates the provisioning of:
•	VPC with public and private subnets
•	Security Groups for ALB, EC2, and database
•	Frontend and Backend Auto Scaling Groups with Launch Templates
•	Application Load Balancers (ALB) for frontend & backend
•	RDS MySQL database for backend
•	Automatic deployment of frontend (Nginx) and backend (Apache + PHP) from GitHub
Free Tier compatible (t3.micro instances).
